ipython notebooks used in section

To use these notebooks, you'll need to grab the files from Github. To do this, you can either fork the repo (if you don't know how to use Github, check out these guides: Getting started with Github, Forking repos), or you can individually download each file. I would personally prefer the first method, but if you don't feel like learning the Github workflow, then this would be the way to go. Just make sure that when you do this, you download every file that's in the folder for the week you're interested in. Then, you can upload these files to datahub (I would recommend creating a new folder to dump these files into; you can make a new folder by clicking the "New" dropdown in the top right corner and selecting the "Folder" option. If you don't have accesss to datahub, you'll have to do this through a local installation of Jupyter. I'd recommend using Anaconda for this; the process for setting this up is detailed extensively in this Piazza post.

Rubik's Cube DeCal:

OLL:

Refer to this page for OLL algorithms.

To get to this point, remember that application of the algorithm F(RUR'U')F' is necessary.

BEGINNERS: Memorize the second algorithm under "All Edges Oriented".

ADVANCED: Memorize all algorithms under "All Edges Oriented".

PLL:

Refer to this page for PLL algorithms.

BEGINNERS: Memorize Aa under "Opposite Edge Swap".

ADVANCED: Memorize all algorithms under EPLL, as well as Aa under "Opposite Edge Swap" and E under "Diag corner swap".
